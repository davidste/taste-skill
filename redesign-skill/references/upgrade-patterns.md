# Upgrade Patterns

Use this file after the audit to replace weak sections with stronger ideas.

## Low-risk wins

- Swap the font pairing before touching layout.
- Tighten the color palette to one accent and one neutral family.
- Add hover, pressed, focus, loading, empty, and error states.
- Add a real container rule and consistent vertical spacing.
- Replace generic shadows with softer, tinted depth cues.

## Layout replacements

- Turn equal feature cards into a zig-zag or split-screen sequence.
- Turn dense dashboard cards into grouped sections with dividers and whitespace.
- Turn a static hero into an asymmetric hero with a product artifact.
- Turn a four-column footer farm into a smaller, intentional footer.

## Motion replacements

- Replace linear easing with spring-based motion.
- Replace instant list or tab switches with shared-layout transitions.
- Replace circular spinners with shape-matched skeletons.
- Replace dead surfaces with subtle staggered entry or one ambient motion cue.

## Surface replacements

- Add restrained glass or inner-border treatments where a premium surface helps.
- Add background texture or image overlays to flat sections.
- Use highlighted rails, tags, or panels instead of stacking more cards.
