# Pattern Library

Use this file when the interface needs a stronger concept. Pick one or two ideas that fit the product; do not stack everything at once.

## Hero directions

- Split-screen hero with copy on one side and a product artifact on the other
- Offset editorial hero with oversized type and generous negative space
- Background-image hero with a controlled fade into the page surface
- Product-first hero where the UI artifact leads and copy supports it

## Layout ideas

- Bento grid with mixed spans and varied surface depth
- Zig-zag feature sections instead of equal columns
- Masonry-style gallery or testimonial wall
- Sticky stacking sections for narrative marketing pages
- Horizontal strip or marquee for dense proof points

## Components and surfaces

- Frosted glass panel with inner border and restrained shadow
- Spotlight border card for hoverable promotional content
- Toolbar or floating command surface instead of a permanent sidebar
- Squircles or rounded-square avatars instead of default circles
- Divider-led dashboard sections instead of card forests

## Motion ideas

- Staggered reveal with spring easing
- Typewriter or shimmer only when the product concept supports it
- Shared-layout transitions for filters, tabs, or bento rearrangement
- Floating status detail or toast that appears with a short overshoot
- Subtle perpetual motion in a tiny isolated component, never across the full page tree

## Dashboard bento prompts

Use these when a SaaS layout needs a premium centerpiece:

1. Intelligent list that reorders itself with smooth layout transitions
2. Command input with typed prompt states and a blinking cursor
3. Live status card with a breathing indicator and timed notification chip
4. Wide metric stream that loops horizontally without visible seams
5. Focus card that highlights content and then reveals a contextual toolbar
